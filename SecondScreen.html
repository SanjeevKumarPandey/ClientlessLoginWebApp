Clientless Login Web App v1.0

Activation Screen





HOW TO USE:

Enter ${REGCODE}, and ${REQUESTOR ID} hit Submit (registration record is fetched). If record is available -> MVPD Picker is created. Pick the MVPD (${mso} id is set); click Login to send the authenticate call (browser handles redirects & cookies). Set this html page as redirect_url by spoofing the localhost to adobe.com. If you do not enter a deviceId and User-Agent, default will be set. Check default by calling appConfig;
